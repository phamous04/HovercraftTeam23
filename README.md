# 🚁 Autonomous Hovercraft Navigation System

> Team engineering project focused on building an autonomous hovercraft capable of navigating a maze and stopping precisely under a target bar using embedded systems and real-time control.

---

## 📌 Project Overview

This project demonstrates the design and implementation of an **autonomous hovercraft** controlled by an **ATmega328P microcontroller**. The system uses multiple sensors to perceive its environment and make real-time navigation decisions while maintaining stable hover, steering, and thrust control.

The hovercraft successfully navigates a maze, avoids obstacles, and stops accurately at a defined end condition.

---

## 🎯 Key Objectives

- Autonomous maze navigation
- Obstacle detection and avoidance
- Stable hover and directional control
- Precise stopping at the target bar
- Reliable real-time performance on constrained hardware

---

## 🧠 System Architecture

The hovercraft uses a modular embedded architecture:

- Microcontroller executes real-time control logic
- Sensors continuously provide environmental data
- Independent lift, thrust, and steering mechanisms
- Continuous sensor processing to drive navigation decisions

---

## 🔧 Hardware Components

### Microcontroller
- ATmega328P

### Sensors
- Ultrasonic sensors
- Infrared sensor
- MPU6050 IMU

### Actuators
- Lift fan
- Thrust fan
- Servo motor for steering

### Power
- Dual battery configuration
